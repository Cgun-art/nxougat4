# nxougat4
a console server client for XBOX one &amp; series, where it has worlds converted to servers, and wii-u nxougat technology. yet it is completely open source &amp; C#/HLSL/GLSL based. due to it having no license, brought to you by some folks, a template for 2b2t &amp; mineville too! ♥️

# How to setup nxougat4:

```lin
neofetch

```

```linux4.6vt

git ///@@@ neofetch

```

```neofetch
in nxougat4 nxougat

```

```pumpkin MC or aternos

c = server
```


# how to run it

use and HDMI television or an xbox one to run the server. :

```
HDMI xbox = true antv xbox XBOXONE 3.88NLC
```

and execute it forever (until the integer time limit is reached:)

```
time crash = @240 2147483647
after crash = restart
```
